# Energia Limpa – Aplicativo Educacional sobre Consumo Energético (ODS 7)

## 1. Descrição do Problema
Muitas pessoas não entendem quanto os aparelhos domésticos realmente consomem e como isso afeta a conta de luz mensal. A falta de clareza sobre conceitos como Watts, kWh, tempo de uso e tarifa energética dificulta decisões conscientes, o que leva ao desperdício e ao aumento do custo de vida. Além disso, o desconhecimento sobre práticas de economia energética contribui para o uso ineficiente de recursos naturais.

---
## 2. Justificativa
O app foi desenvolvido como uma ferramenta simples e acessível para ajudar usuários a compreender o consumo de energia dentro de casa, alinhando-se à ODS 7 – Energia Acessível e Limpa, que busca garantir energia confiável, sustentável e acessível para todos.
Ao oferecer cálculos automáticos, dicas e informações claras, o aplicativo incentiva o uso consciente, reduz desperdícios e promove educação energética, contribuindo socialmente para hábitos mais responsáveis.

---
## 3. Público-Alvo
O aplicativo é voltado para:
- Usuários residenciais que desejam entender sua conta de luz.
- Estudantes que querem aprender sobre consumo energético.
- Pessoas buscando reduzir gastos mensais com energia.
- Famílias interessadas em adotar hábitos mais sustentáveis.

---
## 4. Objetivos do App
### Objetivo Geral
Ajudar o usuário a compreender e controlar melhor seu consumo energético, promovendo conscientização e práticas sustentáveis alinhadas à ODS 7.
### Objetivos Específicos
- Explicar conceitos básicos de energia (Watts, kWh, tempo de uso, tarifa).
- Permitir o cálculo do consumo de aparelhos domésticos.
- Mostrar quais equipamentos gastam mais energia em uma residência.
- Oferecer dicas práticas de economia energética.
- Educar o usuário sobre a importância da energia limpa e eficiente.

## 6. Instruções de Instalação e Uso
### Instalação no Celular (Android)
1. Baixe o arquivo APK disponibilizado na pasta release/ do repositório.
2. No celular, habilite a opção Instalar apps de fontes desconhecidas.
3. Abra o APK e siga o processo padrão de instalação do Android

### Instalação e Execução no Computador (modo desenvolvimento)
Você também pode rodar o app diretamente no computador para testar, modificar ou visualizar o projeto sem usar um celular.

#### Requisitos
- Flutter SDK instalado (versão 3.x ou superior)
- Dart SDK (já vem com o Flutter)
- Android Studio ou VS Code
- Emulador Android OU um dispositivo Android conectado por USB
### Opção 1 — Rodando pelo Android Studio
1. Baixe o repositorio deste projeto.
2. Abra o Android Studio.
3. Clique em Open Project e selecione a pasta do app.
4. No canto superior direito, selecione um emulador Android ou dispositivo físico.
5. Clique no botão Run ▶. 
O app será instalado automaticamente no emulador e executado.
### Opção 2 — Rodando pelo VS Code
1. Baixe o repositorio deste projeto.
2. Abra o VS Code e selecione a pasta do projeto.
3. Certifique-se de que o plugin Flutter está instalado.
4. No rodapé, selecione um emulador Android ou dispositivo físico.
5. Aperte F5 ou clique em Run → Start Debugging.
O app abrirá no emulador conectado.
### Opção 3 — Rodando pelo Terminal
1. Baixe o repositorio deste projeto.
2. Abra a pasta do projeto.
3. Rode o comando:
```bash
flutter pub get
```
4. Depois:
```bash
flutter run
```
Pronto! O Flutter irá detectar um dispositivo conectado ou emulador ativo e iniciará o app.
## Observações:
Caso ocorra algum erro ao iniciar o aplicativo no modo desenvolvimento, basta execultar o seguinte código no terminal dentro da pasta do projeto:
```bash
flutter clean
```
Após isso, só iniciar o projeto normalmente.
